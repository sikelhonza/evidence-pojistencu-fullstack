# Evidence pojištěnců - Fullstack aplikace

Webová aplikace pro správu evidence pojištěnců a jejich pojistných smluv.

## Technologie
- **Backend**: Django + Django REST Framework
- **Frontend**: React
- **Databáze**: SQLite (pro vývoj)

## Funkce
- Přidávání, editace a mazání pojištěnců
- Evidence pojistných smluv
- Vyhledávání a filtrování
- Autentizace uživatelů
